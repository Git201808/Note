## Java语言的特点

1. 跨平台性		平台（不同厂商的芯片）	平台（不同版本的操作系统）
2. 面向对象
3. 简单性   省去了C++多继承，指针等等
4. 健壮性（鲁棒性）   垃圾回收机制  异常处理机制
5. 多线程   并行操作  提高执行性能  线程安全
6. 大数据开发
